# ZM-AI Chat

A lightweight, local or Render-deployable AI chat built using:
- FastAPI (Python backend)
- React + Vite (frontend)
- SQLite (conversation history)
- OpenAI GPT-4o-mini (as backend model)

## Running locally
1. Start backend:
   ```bash
   cd backend
   pip install -r requirements.txt
   python server.py
